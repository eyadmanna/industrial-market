# سوق العدد الصناعية

موقع متكامل لإدارة وعرض سوق العدد الصناعية، مع لوحة تحكم كاملة لإدارة المحتوى.

## المميزات

### الموقع العام
- عرض شرائح متحركة (Slider)
- عرض أقسام السوق (14 قسم) مع صور
- معرض صور متحرك
- خريطة تفاعلية للموقع
- نموذج تواصل مع إشعارات
- تصميم متجاوب (Responsive)
- دعم كامل للغة العربية

### لوحة التحكم
- إدارة الشرائح (إضافة - تعديل - حذف - ترتيب)
- إدارة أقسام السوق مع رفع الصور
- إدارة معرض الصور
- عرض رسائل التواصل
- إعدادات الموقع (الهاتف - البريد - العنوان)

## المتطلبات
- PHP >= 8.1
- Composer
- MySQL >= 5.7
- Laravel 10.x

## التثبيت السريع

```bash
# 1. نسخ المشروع
git clone https://github.com/eyadmanna/industrial-market.git
cd industrial-market

# 2. تثبيت الاعتماديات
composer install
npm install && npm run dev

# 3. إعداد البيئة
cp .env.example .env
php artisan key:generate

# 4. إعداد قاعدة البيانات (عدل ملف .env أولاً)
php artisan migrate
php artisan db:seed

# 5. إنشاء رابط التخزين
php artisan storage:link

# 6. تشغيل السيرفر
php artisan serve
```
